## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session
- createerror

## devDependencies
- Chai
- Mocha
- Nodemon

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js` command.# tinyapp
